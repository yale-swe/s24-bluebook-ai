\n \n Navigate to all answers for Respondent Id 1718256\n 1718256\n \n 1718256\n Absolutely! If you are a computer science major with a mathematical background and are interested in exploring theoretical computer science, this is a great starting point. The theory of computation really only became an independent field in the 20th century, so it's really cool to be learning relatively new concepts. The first half of the class begins with a broad overview of automata/formal language theory, computability theory, and complexity theory, which is followed by coverage of some specialized topics such as the theory of quantum computing, probabilistic methods in computability theory, provability/logic, and more. I'm fascinated by how many fields of mathematics come together in the theory of computation, and it makes me want to take more math classes so I can keep exploring it further. You should have a background in proof-based mathematics, however, before taking this course. The class also moves pretty quickly so having some exposure to some basic models of computation like DFAs and Turing machines is helpful. The problem sets can sometimes be pretty challenging, especially the last one, but this makes sense given that the class has no exams.
