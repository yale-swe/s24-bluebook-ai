\n \n Navigate to all answers for Respondent Id 1621620\n 1621620\n \n 1621620\n There are no weaknesses. \n \n Navigate to all answers for Respondent Id 1621883\n 1621883\n \n 1621883\n Strengths: This is a project-based class first and foremost. If you want to work on something and get a hands-on experience, this class is perfect for you. If you've never used a computer-aided design software, coded, or used arduino, then this is great, but if you have, then a lot of the teaching section of the class will be review. I didn't feel like we really "explored" different fields of engineering much, and having done a fair amount of coding and some modeling on computers in the past, I didn't learn very much. Working on the project taught me valuable skills and was very rewarding, but it can also be time-demanding.
