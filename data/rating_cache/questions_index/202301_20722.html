\n \n Navigate to all answers for Respondent Id 1673635\n 1673635\n \n 1673635\n This course is great! \n \n Navigate to all answers for Respondent Id 1679662\n 1679662\n \n 1679662\n Strength: a great course that lets you explore and get experience with game development.\r\nWeakness: its a little unguided, I wish the class was a little more involved.
\n \n Navigate to all answers for Respondent Id 1679880\n 1679880\n \n 1679880\n Strengths: fun class in a topic that is interesting. Mostly work on projects with a high degree of student control.\r\n\r\nWeaknesses: A bit disorganized, and I wish we went more in depth on game design (looking at specific games or genres).
\n \n Navigate to all answers for Respondent Id 1679891\n 1679891\n \n 1679891\n Strengths: hands-on, gets you think in an innovative and entrepreneurial mindset, teaches you rudimentary knowledge of art and game. Professors are very nice and teaches everything in a way everyone can understand!\r\nWeaknesses: difficult to do the last game project that involves coding. I wish that we were given more time and more instructions on how to code certain things/what we can't learn to code if we did not have previous knowledge coming in. Some of the class had a lot of coding experience, but some of us had no experience at all. I wish the class gave the opportunity to the latter to learn more coding while keeping in mind what we can't learn and apply in a few weeks.
\n \n Navigate to all answers for Respondent Id 1681224\n 1681224\n \n 1681224\n I loved the last 2 assignments (card game and prototype) and the audio walk was also very fun, but I kind of wish we got to start making games earlier in the class, maybe even cutting the audio walk for another game creating assignment (even though I did enjoy the audio walk a lot). The in class gameplay demos were nice where we played and analyzed existing games, while other activities like some of the readings and analysis of stuff like color theory was definitely valuable, but I feel like the more time we spent making games and playing each others games the better. Overall I really liked everything we did but since time is limited I would have liked to have a bit more of the game creation/learn by doing.
