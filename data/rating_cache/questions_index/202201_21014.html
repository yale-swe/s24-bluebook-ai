\n \n Navigate to all answers for Respondent Id 1568068\n 1568068\n \n 1568068\n This course is taught in a lecture format with powerpoints, which is not very conducive for learning a practical skill like music production. Often times lectures felt long and unengaging, and the powerpoints were not helpful. A simple way to make the course so much better would be to do away with the useless powerpoints and to cut straight to showing us how to implement the skill in Ableton Live. It would also help a lot of the demo Ableton files used in class were not already fully completed. Seeing audio effects implemented and adjusted step by step would make learning them so much clearer. I also felt that the assignments were uninspiring and tedious due to over-specification of instructions. If the assignments had a bit more freedom for us to be creative in making our own music, they would have been fun instead of tedious. The only assignment where I really learned a lot was our final project, which was completely open-ended - make a tune or remix in Ableton.

\n \n Navigate to all answers for Respondent Id 1595474\n 1595474\n \n 1595474\n I wish we were given more freedom to explore and exercise creativity in the assignments for this class. I don't see why we couldn't have the current final project as a midterm project, and get students started with making music as soon as possible, so that by the time the final project rolls around, they have some experience under their belts. \n \n Navigate to all answers for Respondent Id 1595474\n 1595474\n \n 1595474\n Yes if you are looking for a gut music class. No if you actually want to learn anything. I was very excited for this class when I first saw it on coursetable - it looked almost perfect on paper. But in reality, this class was pretty disappointing. I learned almost nothing in this class. Granted, I have some experience with music production, but I think I would have learned 10x as much by spending that time making beats on my own. In lectures, Dr. A would demonstrate some of the effects that are possible in Ableton, but we were never given many opportunities to actually apply that knowledge in our own songs, since almost all of the assignments were about matching a template by following a step by step list of instructions (eg. set the velocity of this note to 70) and had no room for creative input. I took CPSC 035 my freshman year and learned so much more in that class than in this one, despite the fact that it was a first-year seminar with no prerequisites, and this class is an upper-level music elective *with* prerequisites. All in all, I felt like this class was extremely hand-holdy, and the expectations for students were so low that we were never really given the opportunity to learn anything. And it's such a shame, because I know some of my classmates are extremely talented musicians, and this class could have been so cool if we were simply allowed to exercise our creativity and run wild with our imaginations. Music production is one of those skills that you can only get better at by practicing a lot, and in this class, were were really only given one single opportunity to do so, and that was the final project. The best part about this class, by a long shot, is getting access to the music studio on the 4th floor of Stoeckel Hall. Everything else was pretty pointless in my opinion.
