\n \n Navigate to all answers for Respondent Id 1704383\n 1704383\n \n 1704383\n In this course you learn Spanish by using it, not by being taught it directly. I think that's a great approach for a few reasons. You find yourself learning without even really trying. It's just through exposure. Of course exposing yourself requires effort, but it's easier to learn a new grammatical structure by seeing it and practicing it with material that is genuinely interesting. The main advantage of this approach is that it's much less exhausting to attend class when you're studying a topic using Spanish, as opposed to studying Spanish by using a topic. There were days it felt like an art history class and not a Spanish class, and those days kept me from feeling burnt out at any point in the semester, which is pretty remarkable for a class that meets 4 days a week +asynchronous day.
\n \n Navigate to all answers for Respondent Id 1705114\n 1705114\n \n 1705114\n The course built well throughout the week and the asynchronous day was a good breakup of information to practice other skills.

The altered version in how this class was taught (emphasis on experiences and participation in class and not so much grades) was excellent and really made me look forward to attending class. To me this class was perfect.
