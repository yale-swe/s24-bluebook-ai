RespondentAnswer
\n \n Navigate to all answers for Respondent Id 1698296\n 1698296\n \n 1698296\n Great opportunity for students to develop themselves as engineers. although, the short time frame for some of the assignments made completing them a little difficult.
\n \n Navigate to all answers for Respondent Id 1700144\n 1700144\n \n 1700144\n This class was essentially an excuse to build something with your team. While we were assigned frequent Engineering Design paper/spreadsheet work, these assignments were trivial to complete, and the real challenge came from actually fabricating the project.\r\n\r\nNot much guidance was provided on how to fabricate your projectafter all, this is not an analysis-based class. I think that more basic engineering principles (basic electronics, basic coding, basic mechanisms) could have been taught before we were to design the projectswith that being said, every team in the class came out with a coherent project, mostly because everyone had prior experience with mechanisms, electronics, and code.\r\n\r\nI don't think the mechanical design processes we were taught helped in any way with our projects. I felt like the task was to build a project, and the "side-task" was to fill out the various mechanical design worksheets. I don't think the worksheets actually influenced the design of my final project.
\n \n Navigate to all answers for Respondent Id 1703925\n 1703925\n \n 1703925\n I think this course is awesome. I think there should be more a design check point where you have to have your fabrication complete halfway through the course. This does not include the wiring just the machining. The course tends to load itself super heavily for the end and this would help alleviate that problem.\r\n\r\nI also wouldn't do cubesats. The class is a mechanical design class not a EE class. It felt a bit like an EE class. I'm not exactly sure what other classes have done as project themes but the cubesat project made it so there wasn't as much specifically mechanical design in the class.
\n \n Navigate to all answers for Respondent Id 1705379\n 1705379\n \n 1705379\n Strengths: Cool project, wonderful TAs\r\n\r\nWeaknesses: There is essentially no teaching in this course relating to mechanical design. We are supposed to find out how to design on our own with our teams, which is not awesome for an intro course where no background is assumed. Doing that was stressful. Also, teams of 3 are hard to navigate, groups of 4 and above should be assigned. The idealization phase was very long, and building was concentrated to the end of the course. It felt like we only had one try to get everything working.
\n \n Navigate to all answers for Respondent Id 1708769\n 1708769\n \n 1708769\n This course is great for learning how real engineering projects work, but the workload goes from nothing to super heavy, all-nighters quickly if you arent super intentional about spreading out your work.
\n \n Navigate to all answers for Respondent Id 1712311\n 1712311\n \n 1712311\n The strengths of the course can mostly be found in the project and the experience it provides. Some of the lessons are also very helpful, as well as the introduction to the machine shop. Weakness-wise, I think some of the lessons are hindered by having to cover too much material in too little time, which can make the deliverables/HW assignments connected to them feel disconnected from the actual learning and engineering process.
\n \n Navigate to all answers for Respondent Id 1712785\n 1712785\n \n 1712785\n The project assignment we were given, making a Cubesat, was very cool. I also thought that the feedback we received on our design from the Nasa engineers was interesting. The schedule for class assignments was put together very well and all 3 of the design reviews and evaluations were helpful in improving the cubesats functionality from a 3d printed prototype to a working prototype with various motors and actuators. I thought that the Failure mode analysis was particularly helpful for choosing parts as well\r\n\r\nI thought that the students could have been given more of a chance to learn how the motors and actuators work/ experiment with them using arduino as well as spend more time with 3d printing before incorporating those motors in the final design project.
\n \n Navigate to all answers for Respondent Id 1715843\n 1715843\n \n 1715843\n I thought that this course was a good introduction to the skills necessary of a successful mechanical engineer. I think that this course could be improved by more effectively using the lecture class periods at the end of the semester to help with the project process more specifically. A class period or two spent on electronics or programming would have been very helpful.
\n \n Navigate to all answers for Respondent Id 1718333\n 1718333\n \n 1718333\n It could be more helpful to have help with time management because our semester long project had a rush at the end to finish.
\n \n Navigate to all answers for Respondent Id 1720262\n 1720262\n \n 1720262\n The course is pretty streamline and the syllabus stated the flow of the course. I would say though that the time during lecture can be used for teams to brainstorm and work, possibly as an additional lab time.
\n \n Navigate to all answers for Respondent Id 1720536\n 1720536\n \n 1720536\n This class provided a great chance to work hands-on, but its weaknesses lie in its lack of structure and instruction. However, the machining was a great introduction in this class.
\n \n Navigate to all answers for Respondent Id 1724378\n 1724378\n \n 1724378\n The strenghts are the professor - Rebecca is very committed to her students' learning and was very approachable and helpful. The main component of the class is definitely the project - which granted was very fun and we did learn a lot from it. The class did also involve getting trained in the machine shop which is a great strength!
