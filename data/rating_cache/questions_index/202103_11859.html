The final project was a good way to exhibit understanding of the material.\r\n\r\nWeaknesses: A lot of the course was spent learning how to implement things like integration and derivatives from scratch, when in reality, Python packages can do the same more quickly and efficiently. The only weakness was that the course felt a bit rushed towards the end, as we were completing a problem set at the same time as the final project with some last-minute directions.
