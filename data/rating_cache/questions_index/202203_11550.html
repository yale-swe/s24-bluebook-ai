\n \n Navigate to all answers for Respondent Id 1609278\n 1609278\n \n 1609278\n Strengths: spectacular class engagement, great feedback on essays, brilliant and friendly instructor. Papers are very efficiently spaced out, and though they are somewhat long there are only four (a fifth optional). A strength was also the grading system - for each essay (of four), we wrote a first draft (*ungraded*!) that we then got feedback on for the graded final draft. I was able to get a lot of personalized feedback

\n \n Navigate to all answers for Respondent Id 1609278\n 1609278\n \n 1609278\n This course should be a requirement for Yale undergraduates.
