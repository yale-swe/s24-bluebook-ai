\n \n Navigate to all answers for Respondent Id 1703648\n 1703648\n \n 1703648\n I think this course would be improved by having a requirement for coding experience, it was listed as accessible for students without coding experience but I found a large barrier for those of us that were hoping to learn more about how to code in this class before it came to applying it.
\n \n Navigate to all answers for Respondent Id 1703874\n 1703874\n \n 1703874\n It was disappointing that this course states it is designed for people with no coding experience, but the coding module itself is not taught in a way that is beneficial to those with genuinely no experience. Julien Berro is an expert in coding, but does not have insight into just how little people who have no coding experience will understand. Following the coding module, we were given complex code throughout the rest of the class, using functions etc that we were never taught, but expected to be able to use and understand on homework assignments and, particularly, the final project.\r\n\r\nTHAT SAID, I think this course could be an excellent and valuable introduction into quantitative approaches given that either a) coding is added as a prerequisite for this course or b) the reliance on coding is removed from the non-coding modules or c) the coding module at the start of the class is curated for beginners, and made to be much longer to give students a better understanding.

\n \n Navigate to all answers for Respondent Id 1717686\n 1717686\n \n 1717686\n The course is taught by three professors and broken down into well-defined units which was a strength of the course as it was very organized. The "introduction to MATLAB" segment is too simple for coders but too fast for beginners. The Fourier transform segment is advanced (and very engaging for me). But the statistics segment seems too basic. The different segments also don't seem cohesive enough to be in the same class, especially the modeling and large dataset analysis.
\n \n Navigate to all answers for Respondent Id 1718173\n 1718173\n \n 1718173\n The course offers a nice survey to some quantitative approaches used in biology, but doesn't really go deep enough to be helpful. So, we only knew some of the basic functions of Matlab at the end of the semester. It would have been better if we were continuously taught about Matlab throughout the semester so that I would be more comfortable with how to use Matlab beyond the basics (for, while loops, making graphs are some of the basics, but it would have been cool to learn about how image processing works on Matlab) or how to develop a skillset for develop code that can be applied to other languages such as Java or C++.
