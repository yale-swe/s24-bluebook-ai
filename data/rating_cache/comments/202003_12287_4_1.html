2020-Fall PHYS 206L 01 Modern Physical Measurement

Q:

Would you recommend this course to another student? Please explain.

 Highly. This is a gem and I'm glad the dept reorganized Phys 206L into this for our virtual semester. Wayyyy better than Phys 205L.
 This course might be revamped into another physics department elective, in which case it is most definitely worth taking. It teaches valuable tools that will come in handy in any form of computational research, and is definitely a valuable addition to the coursework of any physics-related major.
 Yes, I would highly recommend this course to anyone interested in physics research. You will learn very useful skills that you can then later apply.
 Absolutely! For those who have little or no python experience, the learning curve at the beginning is a bit steep, but by about the fourth lab I would say most students were running on a similar level. If you have some python experience, the class is still helpful as an introduction to certain methods such as Monte-Carlo simulations and machine learning techniques, which are still very important skills for research in the sciences.
 If you're taking it, you need it for the major. Otherwise, not really.
 Yes, the python version was much improved. I took 206 in the spring of last sem and had to drop because it was too much to write such long lab reports. This was MUCH more reasonable in terms of workload for a 0.5 level course.
 I would recommend this course to any student who wants a broad and functional introduction to scientific programing. Python is a great first language to learn and this course introduces you to a lot of advanced topics in a really accessible way. This is a great option for students who only need to learn to program for research and other purposes. However I wouldn't recommend this course as an introduction to CS. The skills gained in this course are very practical and application based and thus doesn't cover a lot of the typical base skills for CS.
 Yes I would! Def the best physics class I've taken and perfect for a virtual format!!!
 I would definitely recommend this course to another student who has to take it for their major. I learned a lot of basic concepts in Python programming for physics, so it was a great introduction to data science and machine learning topics. This is not really a substitute for a computer science class, however, and it felt like the vast majority of the students in this class had already learned how to program and had experience with data science and machine learning. Regardless, the work felt useful and meaningful, and the tasks not overwhelming at all, as compared with other lab courses I have taken.
 I think that the relatively low workload and interesting material covered make this course recommendable to other students. It does not demand a lot of working knowledge about physics, programming, or data science, which makes it even more accessible to non-STEM students. The assignments were not incredibly challenging and both the professor and teaching fellows were always eager to help in any way they could.
 YES, this class is essential to the modern physicist! Numerical methods and machine learning techniques are important tools used in research and across higher science. I truly learned some much this semester. This class really needs to become permanent. Also, Professor Nagai is amazing and definitely the one to take this class with! His excitement permeates the lectures and is contagious. The only caveat is that you should be comfortable with the basics of Python before the class starts; otherwise, you will have to scramble through the first couple weeks to which all basic programming is compressed.
 Yes, I would strongly recommend this course to another student. It was very helpful for improving my coding skills and machine learning knowledge.
